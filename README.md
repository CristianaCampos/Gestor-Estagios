# Gestor de Estágios/Internship Manager
PT - O nosso projeto consiste numa aplicação com intuito de evitar o uso de capas de estágio usadas pela maioria das escolas com estágios profissionais. O projeto consiste numa aplicação Web com os vários setores necessários para compor uma capa de estágio da maneira o mais simplificada e eficaz possível.


EN - Our project is an application to avoid the use of internship covers used by most vocational internship schools. The project consists in a web application with several areas to build an internship cover as simply and effectively as possible.
